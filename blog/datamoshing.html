How to datamosh


  1. download avidemux (im using version 2.7.6 installed using flathub)
  2. open ur file
  3. change vid output from copy to mpeg4 avc (x264)
  4. press configure, go to frame tab
  5. change max consecutive b frames to 0, then under i frames change gop size min to 0 and max to 9999
  6. press ok, then save as a new video file.
  7. close the current video, open the one you just saved.
  8. change video output to copy
  9. click the start of the timeline, press the up arrowto skip to next i frame
  10. delete the frame by: starting selection ("A" button or ctrl+pgup), going allong one frame (right arrow) ending selection ("B" button or ctrl+pgdown) then deleting selection (del button)
  11. repeat for all i frames (up arrow, ctrl+pgup, right arrow, ctrl+pgdown, del)
note: alternatively you can start from the beginning and at every jump cut you can delete that frame, whether it be a p frame or an i frame. this will give a better mosh. make sure u still get all the i frames
  1. save.
  2. profit
